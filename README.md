# 3D Portfolio Website

Full single-page portfolio built on the CSS 3D carousel hero template.

## Setup

Add these images to the project root:

- `avatar.png` — hero portrait
- `photoframe.png` — about section photo
- `img1.png` … `img8.png` — 3D ring + project thumbnails

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Structure

| File | Purpose |
|------|---------|
| `index.html` | All sections: hero, about, skills, work, experience, contact |
| `styles.css` | Warm editorial theme + 3D hero styles |
| `main.js` | Nav scroll, mobile menu, form placeholder |
| `Hero.html` | Original hero-only demo |

## Customize

Edit copy, links, email, timeline jobs, and project images in `index.html`. Tweak colors and fonts in `styles.css` (`:root` variables at the top).
